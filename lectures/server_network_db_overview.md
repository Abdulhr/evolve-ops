# Server, Networking, and Database Concepts Overview

## Server Side

* The **server** contains the **business logic**, which defines how data is processed, validated, and saved.
* This logic interacts directly with the **database** to **store**, **update**, and **retrieve** information.
* Example:

  * When a user submits a form on the frontend, the server processes the data and saves it into the database.
